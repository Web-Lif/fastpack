# FastPack 基于 Webpack 的 React 封装

用来快速创建项目，它只是对 React 项目中的 webpack 进行一些预设


## 环境变量

| 名称        |值                                   | 描述
|---          |-----                               |---------
|SpeedMeasure |`json` \| `human` \| `humanVerbose` | 用来查看启动插件的速度信息 默认 `humanVerbose`


## 配置信息

请查看 `FastPackConfig` 的接口类型
